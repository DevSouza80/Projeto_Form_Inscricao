# 📋 Projeto Form Inscrição

Formulário de inscrição com validação em JavaScript puro, desenvolvido como projeto de aprendizado de desenvolvimento web.

---

## 🖥️ Demonstração

![Preview do Projeto](./assets/img/bg_desktop.png)

---

## 🚀 Funcionalidades

- ✅ Validação de campos vazios
- ✅ Validação de e-mail com Regex
- ✅ Validação de senha com regras (mínimo 8 caracteres, letras, números e caractere especial)
- ✅ Mensagens de erro personalizadas
- ✅ Página de sucesso após cadastro
- ✅ Layout responsivo

---

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript** (puro, sem bibliotecas)

## 🧠 O que aprendi

- Selecionar elementos HTML com `getElementById`
- Adicionar e remover classes com `classList.add` e `classList.remove`
- Criar elementos dinamicamente com `createElement` e `appendChild`
- Escutar eventos com `addEventListener`
- Impedir comportamento padrão com `preventDefault`
- Remover espaços com `trim()`
- Validar padrões de texto com **Regex**
- Redirecionar páginas com `window.location.href`
